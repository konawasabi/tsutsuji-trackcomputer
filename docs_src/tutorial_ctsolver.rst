航空写真から曲線軌道を求める
=========================

背景画像機能で表示した航空写真などから

1. カーソルAのPos., Dir.を曲線軌道の始点に合わせる
2. カーソルBのPos., Dir.を曲線軌道の終点に合わせる
3. Mode を次のどちらかに設定してDo Itをクリックする

   1. α(fix)->β(free), R(free)
   2. α(free)->β(fix), R(free)
